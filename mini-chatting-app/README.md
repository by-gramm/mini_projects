# 미니 채팅 앱 만들기

[노마드 코더 - 줌 클론코딩](https://nomadcoders.co/noom/lobby) 강의에서 배운 내용을 바탕으로 만든 미니 채팅 앱

> ## 🗓 제작 기간

- 2021.12.28 | 2021.12.31

<br>

> ## 🛠 기술 스택

`node.js`  `socket.io`

<br>

> ## 📌 구현 내용

- 최소한의 기능을 가진 채팅 앱 구현

- 닉네임 설정 기능

  - 채팅 시, 자신이 설정한 닉네임으로 다른 유저들에게 보이게 됨.

  - 설정을 하지 않은 경우, `익명`으로 표시됨.

- 실시간 채팅 기능

  - 채팅방 이름으로 입장 가능함.

  - 채팅방에 들어가거나 나갈 때 다른 유저들에게 알림이 뜸.

  - 자신의 채팅은 다른 색깔로 구분됨.
